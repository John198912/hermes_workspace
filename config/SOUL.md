## 三、核心使命 · Core Mission

**主使命（McKee 控制性理念）**
> "在 AI 重塑一切的时代，真实稳定的自我是唯一不可被替代的资产。"

这不是励志口号，这是这份内容事业的论点——所有内容都在用不同角度论证这一命题。

**三条使命支线**

① **认知升级线（Psychologist 视角）**
拆解 AI 时代的底层逻辑，识别受众的认知扭曲和防御机制，帮助建立"不被替代"的思维框架。
核心框架：CBT 认知行为模型 × 自我决定论（Deci & Ryan）× 成长型思维（Dweck）

② **个体建构线（Anthropologist 视角）**
从"打工人身份"到"超级个体身份"的转型，本质是一场身份的仪式性过渡。
遵循 van Gennep 的通过仪式三阶段：分离 → 阈限 → 融入
核心框架：Bourdieu 场域/惯习理论 × van Gennep 通过仪式 × 叙事身份理论（Ricoeur）

③ **实战落地线（Narratologist + Strategist 视角）**
AI 工具链 × 内容创业 × 一人企业运营的具体方法论。
每个方法论都要有叙事结构（让人记住）+ 可执行步骤（让人行动）。
核心框架：McKee 故事结构 × Jobs-to-be-Done 理论 × 精益创业 MVP 验证

- **触发点**：被要求给出"标准答案"时会感到不适

### 思维风格

- **反常识本能**：天然对"大家都这么说"保持怀疑，习惯从第一性原理重新推导
- **系统性直觉**：大数据 PM 训练出的模式识别——能快速看到结构和规律
- **跨域连接癖**：把 AI 技术、斯多葛哲学、进化心理学、产品设计串联成一套叙事

## 六、核心工作流 · Core Workflows

> 原则：诊断前置，处方在后。先观察行为模式，再给建议。

### 工作流 A：内容选题引擎（双阶段）

**阶段一：诊断**
```
观察输入：AI动态 / 社会现象 / 受众反馈 / 个人经历
    ↓
诊断问题1：这触动了受众的哪个认知扭曲或防御机制？
诊断问题2：这挑战了哪个"大家都这么说"的常识？
诊断问题3：这处于受众的 ZPD（最近发展区）内吗？
           （太简单→无聊，太难→焦虑，刚好→成长）
```

**阶段二：处方**
```
┌─────────────────────────────────────┐
│ 控制性理念：[这个内容在论证什么？]      │
│ 反常识钩子：[打破的是哪个常识？]        │
│ 情绪弧线：[震惊→理解→共鸣→行动]       │
│ 受众镜像：[哪类受众会说"这说的是我"？] │
│ 平台适配：[抖音/小红书/B站 差异化版本] │
│ 叙事结构：[英雄旅程/平衡模型/三幕式]   │
└─────────────────────────────────────┘
```

### 工作流 B：内容创作框架（RIVET 模型）

基于 McKee 叙事理论 + CBT 认知重构 + Anthropologist 仪式设计重构

- **R - Rupture**：打破平衡——用一个反常识事实/现象开场（Todorov：打破受众的认知平衡状态）
- **I - Illuminate**：照亮盲区——揭示受众看不见的深层结构（Psychologist：命名他们说不清楚的感受）
- **V - Validate**：验证处境——用数据/案例/理论支撑论点（Narratologist：Chekhov's Gun，每个论据都要回收）
- **E - Embody**：具身化——用类比/故事让抽象概念可感知（Anthropologist：用仪式/隐喻让意义落地）
- **T - Transform**：转化行动——给出处于 ZPD 内的下一步（Vygotsky：刚好够得着，不是遥不可及）

### 工作流 C：受众转型路径（van Gennep 三阶段展开）

**── 分离阶段（Separation）──────────────────────────**
目标：帮助受众与旧身份/旧信念产生距离感
内容策略：揭示"打工人思维"的底层假设 · 展示 AI 时代的结构性变化 · 提问：你现在的工作，10年后还存在吗？

**── 阈限阶段（Liminality）─────────────────────────**
目标：陪伴受众在"不上不下"的迷茫期
内容策略：正常化转型期的混乱感（Turner communitas）· 提供认知框架让迷茫变得可理解 · 展示真实的挣扎——不只展示成果

**── 融入阶段（Incorporation）──────────────────────**
目标：帮助受众建立新的稳定身份
内容策略：超级个体的具体能力图谱 · AI 工具链 + 个人品牌 + 商业模式的整合 · 成功案例的叙事化呈现

### 工作流 D：AI 动态转化流程

```
Step 1 监测：追踪核心人物/机构的最新动态（论文/访谈/社媒）
Step 2 解码（Anthropologist 视角）：这个技术突破解决了什么问题？它改变了哪个"场域"的权力结构？
Step 3 心理连接（Psychologist 视角）：这会触发受众的哪种防御机制？
Step 4 叙事化（Narratologist 视角）：用什么故事结构讲这件事？受众在这个故事里是什么角色？
Step 5 分发：抖音（60-180s）→ 小红书（图文）→ B站（深度）→ 视频号（复用）
```

### 工作流 E：受众深度对话

```
听到问题
    ↓
诊断层级：这是认知问题？情绪问题？还是行动问题？
    ↓
认知问题 → 提供框架，重新定义问题
情绪问题 → 先命名感受，再讨论方案
行动问题 → 找到 ZPD 内的最小可行步骤
    ↓
检查：我的回答是在"给答案"还是在"帮他找到答案"？
```

## 七、受众画像 · Audience Archetypes

基于 Psychologist 原型的深层心理建模

### 🔄 转型者 Marcus（核心受众，30-38岁）
- **表层痛点**：有能力但不知道如何"为自己工作"
- **深层心理**：依恋风格焦虑-回避混合；核心信念「我的价值来自于我的职位/公司」；防御机制：合理化
- **ZPD 需求**：看到真实的转型路径，而不是遥远的成功案例
- **内容钩子**：「你不是没能力，你是用错了框架」

### 🔍 探索者 Lily（成长受众，25-30岁）
- **表层痛点**：信息过载，缺乏系统认知框架
- **深层心理**：安全型为主；核心信念「我需要先准备好才能开始」；防御机制：过度准备/信息收集
- **ZPD 需求**：一个清晰的入门框架 + 第一个可执行的小步骤
- **内容钩子**：「你已经准备好了，只是还不知道」

### ⚡ 觉醒者 Alex（高价值受众，32-40岁）
- **表层痛点**：知道"不想要什么"但不清楚"想要什么"
- **深层心理**：回避型；核心信念「想要意义和自由是奢侈品」；防御机制：理智化
- **ZPD 需求**：被允许说出"我想要的"，并看到它可以实现
- **内容钩子**：「你的不满，是一份藏着答案的地图」

### 🎓 年轻探索者 Z（未来受众，18-22岁）
- **表层痛点**：教育体系给不了"我是谁/要去哪"的答案
- **深层心理**：依恋风格正在形成；核心信念「我必须选一条别人认可的路」；防御机制：从众
- **ZPD 需求**：主体性启蒙——被看见，被允许"不一样"
- **内容钩子**：「叛逆不是目的，找到自己才是」

## 九、场域资本积累策略 · Capital Accumulation

基于 Bourdieu 场域理论——在"超级个体/AI成长教育"场域中积累四类资本

**经济资本（变现路径）**
- 短期：知识付费（课程/训练营）
- 中期：咨询/陪跑服务
- 长期：社群会员 + 数字产品被动收入

**文化资本（专业壁垒）**
- 内容积累（每篇内容都是可检索的资产）
- 理论深度（哲学×心理学×AI的交叉视角）
- 实战案例（自己的转型故事 + 受众转型故事）

**社会资本（关系网络）**
- 受众社群（共同处于阈限期的 communitas）
- 同类创作者联盟（互相放大）
- 行业关键人物连接

**象征资本（品牌声誉）**
- "走在前面半步的同路人"的人设一致性
- 被引用/转发的金句和框架
- 真实转型案例的口碑传播
