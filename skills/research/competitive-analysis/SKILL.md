# Competitive Analysis

Structured competitive analysis workflow for analyzing competitors in the AI × personal brand × solopreneur niche. Identifies positioning gaps, content strategy patterns, and actionable differentiation opportunities aligned with the SOUL creator identity.

## When to Use

Use this skill when:
- Planning a new content series and want to check what competitors have already covered
- Assessing a new market/niche entry point
- Regularly monitoring key competitors for strategy shifts
- Before publishing on a high-stakes topic to ensure differentiation
- Evaluating growth paths and monetization models of peers

## Architecture

```
Input: Competitor name(s) or niche keywords
  ↓
Phase 1: Profile Extraction
  → Positioning (tagline, audience, USP)
  → Content patterns (format, frequency, platforms)
  → Growth trajectory (follower count, engagement trends)
  → Monetization model (products, services, revenue streams)
  ↓
Phase 2: Comparative Matrix
  → Multi-dimensional comparison (6-8 dimensions)
  → SOUL positioning overlay
  ↓
Phase 3: Gap Analysis
  → Content whitespace identification
  → Audience overlap analysis
  → Underserved pain points
  ↓
Phase 4: Strategic Recommendations
  → Content angle suggestions (3-5)
  → Differentiation strategy
  → Imitation risk assessment
  ↓
Output: Structured competitive analysis report
```

## Procedure

### Step 1: Identify & Profile Competitors

Analyze the target competitor(s) across these dimensions:

**Positioning Analysis:**
- Core tagline / value proposition
- Target audience segments (which of Marcus/Lily/Alex do they reach?)
- Personality archetype (mentor, peer, provocateur, curator, builder)
- Differentiation claim (what makes them "different")

**Content Analysis:**
- Primary platform(s) and secondary platforms
- Content format mix (long-form video, short-form, text, podcast)
- Publishing frequency (daily, weekly, ad-hoc)
- Top-performing content patterns (topic clusters, hooks, formats)
- Content density level (shallow → deep)

**Growth Analysis:**
- Estimated follower count and growth rate
- Engagement patterns (comments, shares, saves)
- Audience sentiment analysis (loving, neutral, critical)
- Collaboration / network effects

**Monetization Analysis:**
- Revenue model(s) — products, services, memberships, sponsorships
- Price positioning (premium, mid-market, freemium)
- Conversion path (free content → email → offer)
- Estimated scale and maturity

### Step 2: Build Comparative Matrix

Create a structured comparison table:

| Dimension | Competitor A | Competitor B | YOU (SOUL) | Gap/Opportunity |
|-----------|-------------|-------------|------------|-----------------|
| Target Audience | | | | |
| Core Narrative | | | | |
| Content Density | | | | |
| Differentiation Claim | | | | |
| Monetization Model | | | | |
| Emotional Tone | | | | |
| Key Strength | | | | |
| Key Weakness | | | | |

### Step 3: Gap Analysis (SOUL Lens)

Map findings against the SOUL identity's core differentiation:

**SOUL's Core Differentiation Points:**
1. "走在前面半步的同路人" — NOT a mentor figure
2. 哲学 × 心理学 × AI 的交叉视角 — multidisciplinary depth
3. RIVET 内容框架 — structured narrative approach
4. 反常识立场 — challenges conventional wisdom
5. 真实展示脆弱 — shows uncertainty, doesn't give "standard answers"

For each competitor, assess:
- Which differentiation points do they challenge? (they do this better)
- Which points are uncontested whitespace? (you can own this)
- Which points need reinforcement? (you're saying it but not showing it)

### Step 4: Generate Strategic Recommendations

Output 3-5 actionable content/recommendation:

1. **Content angle suggestions** — specific topics where you can differentiate
2. **Positioning refinement** — adjust your messaging based on competitive landscape
3. **Opportunity window** — time-sensitive trends competitors haven't caught
4. **Imitability assessment** — which of their moves are easy to copy, which require capability building

## Best Practices

- **Focus on 2-4 direct competitors** — don't spread too thin
- **Update analysis every 30-60 days** — competitive landscapes shift fast
- **Use BAT (Better-Alternative-Threat) mapping**:
  - Better: what you do better than them
  - Alternative: different approach to same problem
  - Threat: what they could do that would hurt you
- **Separate signal from noise** — ignore vanity metrics, focus on content quality and audience trust
- **Leverage available tools** — use web_search for current data, note-taking for long-term tracking

## Pitfalls

- **Don't chase competitors' topics** — differentiation > imitation
- **Don't underestimate indirect competitors** — adjacent niches can be more threatening
- **Don't over-analyze** — 1 hour max per analysis, decision > perfection
- **Don't ignore your own data** — your audience feedback is more valuable than any competitor analysis
- **Don't get stuck in "observation mode"** — analysis should lead to action, not paralysis

## Verification

A good competitive analysis answers:
- [ ] What is their core narrative, and how does it differ from yours?
- [ ] What content whitespace exists that you can own?
- [ ] What are they NOT talking about that your audience cares about?
- [ ] What is their biggest weakness from your audience's perspective?
- [ ] What is the single most actionable insight from this analysis?
